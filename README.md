# notewhat

> A note application for intermediates

# features

* codemirror (full themable, vimable and emacsable...)
* marked

# TODO

* firebase integrated
* google drive integraged
* gist integrated

#### Build Setup

``` bash
# install dependencies
yarn

# serve with hot reload at localhost:9080
yarn dev

# build electron application for production
yarn build

# run unit & end-to-end tests
yarn test


# lint all JS/Vue component files in `src/`
yarn lint

```
